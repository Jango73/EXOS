# Documentation Layout

This folder contains the repository documentation source and generated outputs.

## Structure

- `guides/`: maintained reference documents and manuals.
- `design/`: architecture notes, audits, and design-oriented material.
- `work/in-progress/`: active implementation notes and working drafts.
- `work/todo/`: planned work that is not being implemented yet.
- `work/bugs/`: bug investigations and archived debugging material.
- `legacy/`: historical material kept for reference.
- `assets/`: documentation assets and source media.
- `shared/`: shared documentation resources such as templates and stylesheets.
- `generated/`: generated documentation output. Do not edit by hand.

## Generated Kernel Documentation

Kernel API documentation generated by Doxygen is written to:

`doc/generated/kernel/`

The main maintained kernel document is:

`doc/guides/kernel.md`
